El comando uname -a muestra información detallada del sistema operativo, como el nombre del sistema, la versión del kernel, la arquitectura y otros datos técnicos del equipo.

El comando whoami muestra el nombre del usuario que está utilizando actualmente el sistema.

Conocer esta información es útil al administrar un servidor porque permite identificar el entorno de trabajo, verificar qué usuario está ejecutando acciones y facilitar tareas de mantenimiento, configuración y seguridad.
